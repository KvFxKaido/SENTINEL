# SENTINEL Portrait Prompts

Ready-to-use prompts for AI image generation. Optimized for Braille conversion (high contrast, clear faces, dark backgrounds).

**Recommended settings:**
- Aspect ratio: 1:1
- Style: Photorealistic or stylized realism
- Avoid: Busy backgrounds, multiple subjects, text

## BATCH GENERATION TIPS

### For Midjourney
- Add `--ar 1:1` for square
- Add `--style raw` for more realistic
- Add `--v 6` for latest version

### For DALL-E
- Specify "digital art" or "photorealistic"
- Square format is default

### For Stable Diffusion
- Add negative prompt: `blurry, low contrast, bright background, multiple people, text, watermark`
- Use 512x512 or 1024x1024

### After Generation
```bash
# Convert to Braille
python -m src.interface.braille portrait.png 30
```
