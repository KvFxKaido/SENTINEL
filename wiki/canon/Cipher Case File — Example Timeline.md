# Cipher Case File — Example Timeline

> [!abstract] Document Type
> **Type:** Gameplay example / play record template
> **Character:** [[Cipher - Sample Character|Cipher]]
> **Purpose:** Shows how to track character history during play

---

## Overview

This document demonstrates how to maintain a character's timeline during a SENTINEL campaign. It tracks:
- Key decisions and their consequences
- Faction relationship changes
- [[Hinge Moments]] that defined the character
- [[Dormant Threads]] waiting to resurface

## Use Cases

- **Players:** Track your character's journey
- **GMs:** Reference past events for callbacks
- **Writers:** Maintain consistency across sessions

## Format

Each entry includes:
1. **Session/Date** — When it happened
2. **Event** — What occurred
3. **Decision** — What the character chose
4. **Consequence** — What changed as a result
5. **Threads** — Unresolved elements for later
